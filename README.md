# MySDOS 🚀
**A Symmetrical Operating System for the Future.**

> "I'm doing a (free) almost-real operating system (just a hobby, won't be big and professional like GNU) for ARM64 and x64 clones. Symmetrical Architecture."

---

## 🌟 About MySDOS
MySDOS is a hobbyist operating system project built with a core focus on **Symmetrical Architecture**. Developed from Dumai, Indonesia, this project aims to explore the full potential of modern processor architectures, specifically ARM64 and x64.

## 🛠️ Tech Stack & Architecture
- **Kernel:** Developed using Assembly (`boot.s`) and C (`main.c`).
- **Target Arch:** ARM64 & x86_64.
- **Goal:** To create a lightweight, symmetrical environment for modern computing clones.
- **Current Status:** Initial source code upload (core kernel files).
